# Scheduled Transactions

If you already have data in your accounts, you can have Actual propose schedules based on the available data.

The system is very flexible, and you can schedule based on various factors:

- Set to recur or only enter once.- Set for automatic entry into the account register or choose to manually approve entries.- Options for recurring entries for one or multiple specific days of the month.
  - For example, a single schedule can be created for a cell phone plan with multiple users and different payment cycles for each phone. If you have three cell phones that all get paid on different days of the month, each day can be defined in a single schedule.
- Options to determine the frequency of payments, such as every month, every 2 months, every 2 years, etc.

For more details, see [Schedules](/docs/schedules).
