Welcome to Actual's documentation! This page provides you a quick introduction to Actual Budget and will help you navigate the documentation to learn more about Actual.

## Documentation Overview

The main pages are listed in the left sidebar menu (collapsed "hamburger" menu at the top left in mobile devies). Click on a chevron to view more nested pages.

Each page has sections that show in the right sidebar (or at the top under "On this page" in mobile devices) for quick navigation to a specific subject.

An entry in the search bar will show the most useful result at the top, but all results it finds will be available.

## Where to Begin?

If you're new to Actual, but don't know where to start, consult the [roadmap for new users](/docs/getting-started/roadmap-for-new-users.md).

You can also [install Actual](./install/index.md) on a computer to start. After installation, check out the [Tour of Actual](./tour/index.md) above for a
high-level overview of how Actual works. The next step is to read the [Starting Fresh](/docs/getting-started/starting-fresh) guide. Then we suggest you dive into the more detailed documentation.

For help and support, [reach out to the maintainers and community](/contact).

## Want to Contribute?

If you'd like to contribute to Actual Budget's documentation, start by reading the documentation standard, local setup, and guidelines on using images and screenshots from [here](/docs/contributing/writing-docs) and [here](https://github.com/actualbudget/actual/tree/master/packages/docs#actual-budget-community-documentation). We use Docusaurus to build our website. You can click on _Edit this page_ at the bottom left of every page to open that page's github document.
