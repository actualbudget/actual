Allow running DB queries against the local database outside of dev-mode.

Read more in: https://actualbudget.org/docs/api/actual-ql/

Sample:

```javascript
window
  .$query(window.$q('accounts').select('*'))
  .then(({ data }) => console.log(data));
```
